HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Selector defines the element or attribute to which declarations will apply.
  • Declarations contain the css property we want to apply and the value of the property - e.g. color: blue
  • The * selector applies declarations to all elements on the page
  • A Margin indicates how much space we want around the element
  • Padding indicates how much space we want aroudn the content inside the element
    (think padding in a box)

Git

  • git status: checks what branch we are currrently on
  • git checkout "branch name": switched to named branch
  • git checkout -b "branch-name": creates a new branch and switches to it
  • git add -A: stages all (-A) changes that have been made.
  • git commit -m "message": commits all changes that have been staged, so that they may be pushed to the repo, along with a message (-m)
  • git pull origin main: pulls the main file from the repo to ensure up to date files.
  • git push origin "branch name": pushes commited changes to the online repo to be reviewed.

Javascript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A switching statement is a condensed version of a number of if-else statement
  • A Function is a segment of code or instructions that you can reuse many times without re-writing by simply calling the fuctions' name
